=============================
metadata field
=============================

**表示资源的元数据。**

嵌套字段如下：

----------------------
name
----------------------

在同一个namespace中必须保持唯一

----------------
namespace
----------------

资源对象所属的namespace

如果为空则为default

查看当前集群namespace ``kubectl get namespace``

----------------------
labels
----------------------

定义资源的标签，可以被selector（标签选择器）等进行匹配选择。

----------------------
annotation
----------------------

定义资源注解。
